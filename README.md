# aplikasi zero waste

## deskripsi proyek
kami memilih topik sdg 2 (zero hunger) karena ada kontradiksi besar di sekitar kita: di satu sisi banyak orang kelaparan, tetapi di sisi lain banyak makanan layak konsumsi yang dibuang begitu saja dari restoran atau kafe. aplikasi zero waste hadir sebagai solusi digital untuk mempertemukan donatur yang memiliki makanan berlebih dengan masyarakat yang membutuhkan, agar makanan tidak terbuang sia-sia.
